# 자바 온라인 오목 - Omok


<div id="2"></div>

##  개요
- 온라인 상에서 대결할 수 있는 오목 게임
- 서버-클라이언트 Socket 프로토콜 송수신

##  주요 기능

- 오목
- 로그인 기능
- 2인, 3인(1:1:1), 4인(2:2) 배틀 추가
- 방 생성, 입장, 준비, 시작, 종료 로직
- 33 불가, 타이머(30초), 무르기 기능 
- 방 목록, 방 내 유저 목록 리스트로 관리
- 채팅 기능(전체, 게임 내 별도) 
- 방장의 강제퇴장 기능, 채팅금지 기능
- 게임 플레이 외 관전모드(이전 다음으로 진행과정 복기 가능)
- 관전자 훈수두기
- 개인 아바타 선택
